                        function getValue(){
                        
                            //making sure the alert is not visible
                            document.getElementById("alert").classList.add("invisible");

                        //check if string is a palindrome
                        function checkForPalindrome(userString){
                        
                            userString = userString.toLowerCase();
                        
                            //remove spaces and special characters
                            let regex = /[^a-z0-9]/gi;
                            userString = userString.replace(regex, "");
                        
                            let revString = [];
                            let returnObj = {};
                        
                            //reverse the string
                            for (let index = userString.length - 1; index >= 0; index--) {
                                revString += userString[index];        
                            }
                        
                            //checking the string for a palindrome
                            if (revString == userString) {
                                returnObj.msg = "Well Done! You entered a palindrome!"        
                            } else {
                                returnObj.msg = "Sorry! You did not enter a palindrome!"
                            }
                        
                            returnObj.reversed = revString;
                        
                            return returnObj;
                        }
                        
                        // display a message to the string
                        function displayMessage(returnObj){
                        
                            document.getElementById("alertHeader").innerHTML = returnObj.msg;
                            document.getElementById("msg").innerHTML = `Your phrase reversed is: ${returnObj.reversed}`;
                            document.getElementById("alert").classList.remove("invisible");
                        }

The Code is structered a combination of three functions that work with data processed or gathered by eachother.

getValue()

Inside this function dictates what the program does later but first off it sets the alert to invisible. We declare a variable and assign it a value that is taken from the page. Next we go ahead and declare another variable while calling the function checkForPalindrome(). Eventually after checkForPalindrome() returns the data it is then passed into the displayMessage() function.

checkForPalindrome()

This function is responsible for all the logic and ensuring that the user gets their desired output. It accepts a parameter which in this case is a string. The string is immediately has all of its characters set to lowercase with the .toLowerCase() method. The next thing that happens is the use of regex to remove the spaces and special characters that might be in the string. At this point we declare an object and an empty array to be used later. We reverse the string using a for loop and we check the string to see if it is a palindrome using an if statement that compares both strings. Once that is done we then return the data.

displayMessage()

Out of the entire program this is the most simple function out of them all. This function accepts an object and uses it to display the message to the user. It starts off by returning the message that is contained within the object within the "alertHeader". Next it displays to the user their string but reversed. Lastly in order to show all of this data to the user we simply remove the "invisible" class from the alert to display everything on the page.
